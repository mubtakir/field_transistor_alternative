# FTA Virtual Laboratory v1.0
## Unified Physics Simulation Workspace for Field Transistor Research

Welcome to the **FTA Virtual Laboratory**, a professionalized R&D environment for simulating and designing the **Field Transistor Alternative (FTA)**.

### 🔬 Project Overview
This laboratory transitions the project from standalone research scripts into a unified, modular architecture. It separates foundational physics solvers from innovative device designs, providing a robust platform for verifiable scientific research.

### 📂 Directory Structure
- `physics_engine/`: Tier 1 Foundational Solvers (Laplace, WKB, FDTD, Thermal).
- `device_models/`: Tier 2 Applied Innovation (U-Plate FTA, Differential Pair).
- `analyzers/`: Tier 3 Automated Characterization (IV, Frequency Response).
- `lab_manager.py`: Core orchestrator for the laboratory.
- `config.yaml`: Centralized configuration and materials database.
- `examples/`: Ready-to-run demonstration scripts.
- `outputs/`: Automatically organized simulation data and plots.

### 🚀 Getting Started
To verify the environment and run your first simulation:
```bash
python fta_lab/examples/example_basic_fta.py
```
