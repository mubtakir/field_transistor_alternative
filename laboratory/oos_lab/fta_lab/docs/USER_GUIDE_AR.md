# دليل المستخدم: مختبر FTA الافتراضي v1.0
## دليل شامل للباحثين والمهندسين

يقدم هذا الدليل خطوات مفصلة لاستخدام **مختبر FTA الافتراضي** في أبحاث وتطوير أشباه الموصلات المتقدمة.

---

## 1. البداية السريعة: مركز التحكم الرسومي (GUI)
أسهل طريقة للتفاعل مع المختبر هي من خلال **لوحة التحكم** القائمة على الويب.

### كيفية التشغيل:
1. افتح نافذة الأوامر (Terminal) في مجلد المشروع.
2. نفذ الأمر: `python fta_lab/gui/server.py`
3. افتح المتصفح على العنوان: `http://localhost:8000`

### مميزات لوحة التحكم:
- **أشرطة التمرير الهندسي**: تعديل طول الجهاز، عرضه، والفجوة (بالنانومتر) في الوقت الفعلي.
- **تشغيل المحاكاة**: تفعيل محركات الفيزياء لحساب خصائص التيار والجهد (IV).
- **الرسم البياني المباشر**: عرض منحنى تيار المصب (Id) مقابل جهد البوابة (Vg).
- **حالة المواد**: عرض خصائص المواد النشطة حالياً (النحاس، SiO2، Mu-Metal).

---

## 2. الأبحاث المتقدمة: واجهة البرمجة (Python API)
للمحاكاة المخصصة والتحليل الضخم، استخدم واجهة برمجة بايثون.

### مدير المختبر (Lab Manager)
فئة `FTALaboratory` هي المسؤولة عن الإعدادات وإدارة الجلسات.
```python
from fta_lab.lab_manager import FTALaboratory
lab = FTALaboratory()
```

### إنشاء الأجهزة
قم بإنشاء نموذج لمعمارية معينة (مثل `u_plate`):
```python
device = lab.create_device('u_plate', geometry={'length': 100e-6, 'gap': 5e-9})
```

---

## 3. محاكاة بوابات المنطق (Phase 7)
يدعم المختبر نمذجة بوابات المنطق بشكل حيوي.

### مثال بوابة NOT:
```python
from fta_lab.device_models.logic_gates import FTANotGate
gate = FTANotGate({'length': 100e-6, 'gap': 5e-9}, V_DD=5.0)
result = gate.solve(V_in=5.0) # محاكاة مدخل 'مرتفع'
print(f"Logic Output: {result['logic_out']}")
```

---

## 4. تخصيص محركات الفيزياء
بني المختبر على **بنية طبقية**:
1. **الطبقة الأولى (المحركات الفيزيائية)**: في مجلد `physics_engine/`. تشمل حلول Laplace و WKB و FDTD والحرارة.
2. **الطبقة الثانية (نماذج الأجهزة)**: في مجلد `device_models/`. تنظم المحركات لنمذجة جهاز وظيفي.
3. **الطبقة الثالثة (المحللات)**: في مجلد `analyzers/`. أدوات عالية المستوى لتحليل DC و AC.

---

## 5. الإعدادات (config.yaml)
قم بتعديل ملف `config.yaml` لتغيير:
- **الدقة الافتراضية**: دقة حلول FDM.
- **خصائص المواد**: الناقلية، سماحية العزل، والنفاذية المغناطيسية.
- **ملف العتاد**: التنقل بين التنفيذ عبر المعالج **CPU** أو المعالج الرسومي **GPU (Taichi)**.
